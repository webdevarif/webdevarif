## Getting Started

### Prerequisites

- Node.js >= 20.9
- pnpm >= 10

### Installation

```bash
# Clone the repo
git clone https://github.com/webdevarif/webdevarif.git
cd webdevarif

# Install dependencies
pnpm install

# Run dev server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build

```bash
pnpm build
```
